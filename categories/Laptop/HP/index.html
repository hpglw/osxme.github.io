HP HP OMEN2 PRO（OMEN by HP Laptop） Laptop i7-6700HQ HACKINTOSH EFI OC7.4 Big Sur 11.6 DGPU

Hardware Configuration

CPU INTEL Core i7-6700HQ @ 2.60GHz
Motherboard HP 8294 ( 100 Series/C230 Series Chipset Family - A14E )
Memory 16 GB ( Hynix DDR4 2400MHz / KingSton DDR4 2400MHz )
Hard Drive SanDisk SD8SNAT-128G-1006 ( 128 GB / SSD)
GPU AMD Radeon RX 460 ( 4 GB / HP )
Display LG LGD0532 ( 15.5 )
Audio Realtek @ INTEL High Definition Audio Controller
Network INTEL Dual Band Wireless-AC 7265

BIOS里关掉安全启动

这个HP的BIOS 能自动识别OC EFI ，并只能认WINDOWS的 EFI 也就是WINDOS BOOT MANAGER，要用DISKGUEIS或者EASYUEFI新建一条OC的 EFI ，然后进去BIOS系统设置，的启动管理员，把OC的 EFI 按F6移到最上面，记得再按F10保存住才行，
DISKGUEIS或者EASYUEFI建立的OC EFI 指向OPENCORE.EFI。

最神奇的地方在于这台 Laptop 有RX460的独显，是可以完的！！

读卡器可用，WIFI可以用，记得用HELIPORT工具，蓝牙可能 行， IGPU ，Audio （从WIDNOWS重启进MACOS会没声，关机后开机直接进MACOS才有声音），触控板，电池，USB己经定制过了，摄像头可用，变频正常，睿频OK。
