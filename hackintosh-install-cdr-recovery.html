黑苹果之安装版，懒人版，恢复版的区别，优劣比较。

发表于 2019-10-28 更新于 2019-10-29 分类于 教程

目前安装黑苹果主要有几种方法，比较常见的有U盘安装版，CDR格式懒人版和硬盘恢复版。其实跟windows一样，U盘安装就像IOS镜像刻录的U盘从头到尾安装，懒人版和恢复版更像GHOST系统。U盘安装时间长一些，GHOST版就快捷方便，省时间。

但macOS跟windows不一样的是，macOS系统安装之后不会为机子生成单独的配置文件，而是全部文件都会释放出来，也就是说无论安装在什么电脑上，系统文件都是一样的。而windows则会为每台机子生成不同的驱动和注册表信息，这也是为什么windows的系统打包后还原到其它机子上，可能会出现蓝屏，或者硬件无法加载。

现在来讲讲三种安装方法的不同：

1，U盘安装版

一般由官方原版镜像打包成DMG格式，也就是macos的安装镜像格式文件，然后用balenaEtcher一类的刻录工具，将dmg镜像刻录成安装U盘。像网上发布的U盘镜像一般还会自带比较通用的CLOVER四叶草引导，对应不同的配置，有些只要稍为修改一下就能成功安装黑苹果。

  • 好处是绝对原版系统，心理上舒服，强迫症首选，一步步安装下来，能体验一下安装成功mac的快感。
  • 劣处是U盘安装速度较慢，需要重启几次，介质不能出错，由USB端口引发的错误更是比较难解决，比较适合爱折腾的。

2，懒人版

懒人版一般是CDR格式，由安装好macos的机子打包系统生成。懒人版一般现在比较少用了，因为安装步骤还是比较多的，以前在变色龙引导时代比较流行，现在一般是CLOVER了。

  • 好处是可以用MBR磁盘，用硬盘安装助手直接写入，也比较省时间。
  • 劣处是安装步骤太多，更容易出错，的确不怎么建议。

3，恢复版

黑苹果恢复版跟懒人版比较相似，都是不通过U盘安装。同样由安装好的macos整个打包完成，适合双系统，目前淘宝远程安装大多采用这种方法，省时间和精力，只要CLOVER引导正确，直接可以进入系统，更多的时间只是在调驱动。老吴现在就是选用这种方法安装黑苹果。

  • 好处是不用U盘，安装快速，只要打包的镜像没问题，基本是原版系统。
  • 劣处是需要windows系统才能远程操作，别人打包的系统总感觉会有病毒或者其它东西注入了。

综合以上，就是三种不同的安装黑苹果的方法，目前懒人版比较少用，一般就是U盘安装和恢复版。U盘版适合电脑知识比较丰富的，比较喜欢原汁原味的，动手能力强的。恢复版比较适合省时间和精力的，可以更专注于CLOVER引导调试和完美的。

老吴这边还是比较推荐恢复版，可以跳过好多不必要的错误，从而更省时间，只要打包的镜像完整，就不必但心系统是否稳定，再说，系统是否稳定，取决于CLOVER引导。
