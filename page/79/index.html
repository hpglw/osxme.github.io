【台式机】宏碁 Aspire TC-780A i5-7400 HD630 ALC662 10.14.6黑苹果引导_Hackintosh_Clover

发表于 2019-12-30 更新于 2022-08-10 分类于 台式机 ， INTEL

简介

宏碁 Aspire品牌机安装黑苹果mojave 10.14.6。这是一台只有核显HD630的品牌机，需要注意的是，BIOS的限制比较多，必须使用HDMI接口接显示器，否则黑屏。特别注意一点是！！这个BIOS只认windows引导，CLOVER的引导要是放在ESP盘不可能让它启动的，解决方法是要么直接用U盘引导，要么新建一个300MB的FAT32分区，把CLOVER放在里面，这样就可以了。

配置

电脑型号 宏碁 Aspire TC-780A 台式电脑
操作系统 Windows 10 64位 ( DirectX 12 )
处理器 英特尔 Core i5-7400 @ 3.00GHz 四核
主板 宏碁 Aspire TC-780A(KBL) ( 100 Series/C230 Series 芯片组 Family - A143 )
显卡 英特尔 视频控制器 ( 128 MB / 宏碁 )
内存 16 GB ( 芝奇 DDR4 2400MHz )
主硬盘 东芝 DT01ACA100 ( 1 TB / 7200 转/分 )
显示器 明基 BNQ7F4F ZOWIE RL LCD ( 24 英寸 )
光驱 建兴 DVD A DA8AESH DVD刻录机
声卡 瑞昱 ALC662 @ 英特尔 High Definition Audio 控制器
网卡 瑞昱 RTL8168/8111/8112 Gigabit Ethernet Controller / 宏碁
