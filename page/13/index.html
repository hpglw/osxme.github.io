哪些AMD CPU可以支持安装黑苹果？amd黑苹果cpu支持列表

发表于 2020-01-10 分类于 教程

目前，AMD支持的黑苹果的CPU主要是以下几种：

  • 核心架构：Bulldozer 如FX系列
  • 核心架构：Jaguar 如A系列
  • 核心架构：Zen 如Ryzen, Threadripper, Athlon 2xxGE系列

支持的macOS版本：

  • High Sierra 10.13.6 (17G65, 17G66, 17G8030, 17G8037)
  • Mojave 10.14.6 (18G84, 18G87, 18G95, 18G103)
  • Catalina 10.15.0 (19A583, 19A602), 10.15.1 (19B88)

常见的黑苹果AMD CPU有：1700x，AMD Ryzen 5 2600，FX6300，AMD Ryzen 3600g，AMD Ryzen 7 2700X，AMD Ryzen 5 1400，X4-760K,AMD RYZEN 3700x等。

完全支持:

  • Ryzen:

    • Zen:
      • Ryzen 3 1200 至 Ryzen 7 1800X, 所有X系列
      • Threadripper 1900X, 1920X, 1950X
    • Zen+:
      • Ryzen 3 2300X 至 Ryzen 7 2700X 所有X和MAX系列，G系列除外。
      • Threadripper 2920X, 2950x, 2970WX, 2990WX
    • Zen2:
      • Ryzen 5 3500 至 Ryzen 9 3950X, 所有X和MAX系列，G系列除外。

  • 15/16h:

  • 所有核显均不支持黑苹果！必须使用独立显卡*

    • Bulldozer
    • Piledriver
    • Steamroller
    • Excavator
    • Trinity
    • Richland
    • Kaveri
    • Carrizo
    • Bristol Ridge

  • Zen (Raven Ridge):

    • Athlon 200 GE 至 Athlon 240 GE
    • Athlon 3000G
    • Ryzen 3 2200GE 至 Ryzen 5 Pro 2400G

  • Zen+ (Picasso):

    • Athlon Pro 300GE
    • Ryzen 3 3200G 至 Ryzen 5 Pro 3400G

TX40 3000-系 Threadripper

  • Threadripper 3960X
  • Threadripper 3970X
  • Threadripper 3990X

任何AMD CPU的笔记本均不支持黑苹果，不用考虑！

Opteron和EPYC处理都没有经过测试，不确定能否使用。

15/16的CPU可能存在一定的问题，不建议再使用。

请注意的是，AMD 黑苹果为适合于生产环境，可能存在一定的软件和系统兼容性问题，适合一般家用。

双系统不想要黑苹果了，如何删除黑苹果系统教程

发表于 2020-01-09 分类于 教程

很多人用不习惯macos系统，想要删除黑苹果系统，操作方法如下，在双系统环境中，进入windows，有三个地方要操作的，删除黑苹果就很干净了。
需要准备两个工具：
DiskGenius
EasyUEFI
1，首先，打开计算机的管理——磁盘管理


把安装了macos黑苹果的分区整个删掉！然后可以合并，也可以新建成ntfs格式，这个随意。

2，打开DiskGenius工具，找到ESP盘，里面有个EFI文件夹，把EFI文件夹中的CLOVER文件夹删掉。

3，安装EasyUEFI工具，打开找到 CLOVER的引导项，删掉！

就这样，黑苹果就彻度删除了！

黑苹果怎么屏蔽三星PM981

发表于 2020-01-09 分类于 教程

首先，进系统后，第一时间，去启动台——其它

打开磁盘工具，把三星PM981的所有分区都卸载掉先，否则可能会卡死或者自动重启，无法进行后续的屏蔽工作。

1，打开终端，输入diskutil list
2，找到PM981的硬盘信息：

每个人的分区信息，都不一样，我们只需要找到可见的分区，如C盘D盘这些，在INDENTIFIER参数中，如C盘的分区是disk1s1，记下来。
3，diskutil info disk1s1，然后找到Volume UUID值，复制出来
4，再打开终端，输入sudo vifs
5，在vim编辑器中按i，然后在最后一行输入UUID=Volume UUID none ntfs noauto，有几个分区，就有几行。
6，保存vim退出，重启之后，开机自动卸载PM981所有磁盘分区，屏蔽成功。

再谈谈三星PM981和EVO 970 PLUS安装黑苹果的问题

发表于 2020-01-09 分类于 教程

虽然说这两款NVME固态的确性价比不错，很多笔记本，如小米，戴尔，联想都选用了这款硬盘。

PM981型号开头为MZVLB，只要有MZLVB就是PM981，还有PM981A两种。但就安装黑苹果来说，目前为止不可以，PM981和PM981A绝对是不行的。国内有些论坛说己经解决，那是不可信的，因为老吴己经安装太多了，无非表现都是开机能进系统，但会莫机卡死，自动重启等一系列奇怪的问题，反正就是不能正常使用，也用过各种说可以解决的方法，无果！从国外的大神反应来看，都是保持沉默，要么建议更换更换其它硬盘

如果你的笔记本中有PM981，也不是说一定安装不了，一般现在的笔记本都可以安装两个硬盘，只要将黑苹果安装在另一块上面，然后进黑苹果之后屏蔽PM981即可，屏蔽的方法很多，有DSDT法，也有开机不挂载法，老吴认为开机不挂载法是最方便的，也是最简单的
教程可以看：http://k61.org/diable-pm981-on-hackintosh.html

另外还有一个三星EVO 970 PLUS，目前这一块可以通过刷新最新的固件，安装黑苹果是没有问题，教程在这里：http://k61.org/970-EVO-Plus-firmware-flash.html
但是，目前发现了在macos catalina 10.15.2版本中，这块硬盘又出问题了！！再次不兼容！具体表现就是无法进系统，卡在苹果LOGO上，还没有解决方法，只能用10.15.2以下的macos版本。

老吴建议，三星这两款固态的，都不要安装黑苹果，就算安装，也只能在其它硬盘上装，然后还是屏蔽掉，也许只能等固件更新，毕竟三星也不会为了黑苹果而去修改固件，更多的还是为windows系统考虑。
