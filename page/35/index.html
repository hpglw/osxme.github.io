Hackintosh 10.13.6 FCPX剪辑绿屏 闪退 意外退出问题解决思路

这是一台Hackintosh 主机的配置

可以看到 GPU 是QUADRO的K2200，使用WEBDIRIVER驱动，可以 Install macos 10.13.6。

目前遇到的问题是FCPX 10.4剪辑的时候，加上特效，转场，或者防抖动特效时，画面会变成绿点，绿块

有时会闪退，自动退出，意外退出

主要是这两个问题。

解决的思路是

1，查看 GPU 的驱动是否最新，应应用最新的WhateverGreen.kext
2，CLOVER或者OC的版本更新到最新的版本
3，尝试更新 同的仿冒机型
4，FCPX的素材尽量 要使用局域网盘，放在本机 SSD 上。
5，更新最新的Network 驱动。
